接口定义 - 标准 USB：

| 触点 | 功能                 |
| ---- | -------------------- |
| 1    | VBUS（4.75－5.25 V） |
| 2    | D-                   |
| 3    | D+                   |
| 4    | GND                  |

接口定义 - Mini USB：

| 触点 | 功能                 | 颜色 |
| ---- | -------------------- | ---- |
| 1    | VBUS（4.75－5.25 V） | 红   |
| 2    | D-                   | 白   |
| 3    | D+                   | 绿   |
| 4    | ID                   |      |
| 5    | GND                  | 黑   |
